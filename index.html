CSS Inline

Um subtítulo

No CSS Inline, usamos o atributo style dentro da TAG que queremos estilizar no HTML. Isso estilizara um elemento sem a necessidade de criar um arquivo CSS. Essa forma é menos utilizada e recomendada, porque usando essa forma, teremos que estilizar cada elemento separadamente. Se o site conter 20 paragráfos, teremos que adicionar estilo um por um.

A anatomia ficaria dessa forma:

<h2 style="valor";>TÍTULO</h2>
